✈️ Tripify
📌 Projenin Amacı

Tripify, tur ve seyahat sektörüne yönelik geliştirilmiş, kullanıcıların tur seçeneklerini inceleyebildiği, rezervasyon yapabildiği ve seyahat deneyimlerini paylaşabildiği yapay zeka destekli bir web uygulamasıdır.

Bu proje; modern web uygulamalarında kullanılan katmanlı mimari, REST API yapısı, yapay zeka entegrasyonu ve rezervasyon yönetimi gibi süreçleri gerçek bir senaryo üzerinden uygulamalı olarak deneyimlemek amacıyla geliştirilmiştir.

Tripify sayesinde kullanıcılar farklı destinasyonlara ait tur seçeneklerini inceleyebilir, tur detaylarını görüntüleyebilir, rezervasyon oluşturabilir ve daha önce bu turları deneyimleyen kullanıcıların yorumlarını okuyabilir.

Proje içerisinde kullanılan yapay zeka entegrasyonları, tur detay sayfalarında kullanıcı deneyimini zenginleştiren içerikler üretmektedir. Yapay zeka; tur hakkında günlük gezi planları oluşturmak, kullanıcı yorumlarını analiz ederek özet çıkarmak ve rezervasyon işlemleri için e-posta içerikleri hazırlamak gibi işlevleri yerine getirmektedir.

Bu sayede sistem yalnızca bir rezervasyon platformu değil, aynı zamanda kullanıcıların seyahat planlarını kolaylaştıran akıllı bir seyahat asistanı görevi görmektedir.
🗄️ Veritabanı
MongoDB

Projede veri depolama işlemleri için NoSQL veritabanı olan MongoDB tercih edilmiştir.
Tur bilgileri, rezervasyonlar, kullanıcı yorumları ve diğer sistem verileri MongoDB üzerinde saklanmaktadır.

📄 PDF Oluşturma

Rezervasyon bilgileri gerektiğinde PDF formatında dışa aktarılabilmektedir.
Bu özellik sayesinde kullanıcılar rezervasyon bilgilerini belge olarak indirebilir veya saklayabilir.

🌍 Çoklu Dil Desteği

Projede kullanıcı deneyimini artırmak amacıyla yerelleştirme sistemi kullanılmıştır.

Bu yapı sayesinde uygulama farklı ülkelerden kullanıcılar için kolayca kullanılabilir hale getirilmiştir.
🤖 Yapay Zeka Entegrasyonları

Projede kullanıcı deneyimini geliştirmek amacıyla OpenAI Chat Completions API kullanılmıştır.

🗺️ Yapay Zeka Destekli Gezi Planı

Tur detay sayfasında yer alan “Neler Bekleyebilirsiniz” bölümünde, yapay zeka tarafından oluşturulan günlük gezi planları gösterilmektedir.

Yapay zeka;

Tur başlığı

Tur açıklaması

Tur lokasyonu

Tur süresi

bilgilerini analiz ederek 4 günlük detaylı bir seyahat planı oluşturur.

Bu plan içerisinde:

Günlük gezilecek yerler

Yapılabilecek aktiviteler

Yemek önerileri

gibi bilgiler kullanıcıya sunulmaktadır.
🚀 Projede Gerçekleştirilen İşlemler

Bu proje kapsamında aşağıdaki temel sistem akışları başarıyla uygulanmıştır:

✔ Tur Listeleme
Kullanıcılar sistemde bulunan tüm turları görüntüleyebilir.

✔ Tur Detay Sayfası
Seçilen tur hakkında detaylı bilgiler ve gezi planı görüntülenebilir.

✔ Rezervasyon Oluşturma
Kullanıcılar seçtikleri tur için rezervasyon oluşturabilir.

✔ Kullanıcı Yorumları
Tur deneyimi yaşayan kullanıcılar yorum ve puan bırakabilir.

✔ Rezervasyon Yönetimi
Admin paneli üzerinden rezervasyonlar onaylanabilir veya reddedilebilir.

✔ E-posta Bildirimleri
Rezervasyon durumuna göre kullanıcıya e-posta gönderilir.

✔ Admin Paneli Yönetimi
Admin kullanıcıları turlar, rezervasyonlar ve yorumlar üzerinde yönetim işlemleri gerçekleştirebilir.
